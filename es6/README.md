### 技术栈

- ES6: 这是以后的趋势，let、cons、箭头函数、Set Map 数据结构、Promise 等新特性十分实用
- Gulp: 编写 gulp 脚本完成ES6的自动编译、打包、文件监听、浏览器热更新、模板自动更新、服务热启动等
- Babel: 解决浏览器兼容es6问题，实现了 ES6 转 ES5
- Webpack: 自动编译
- Express.js: 搭建服务器

### Build Setup
```
# run gulp
$ gulp --watch

# visit
$ http://localhost:3000/
```
